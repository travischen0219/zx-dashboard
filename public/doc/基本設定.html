在每個 table  裡


是否每個table 要有 新增/修改/刪除  的日期時間和人員 


在程式裡有寫入到資料庫 的動作：

1.如果是新增   寫入資料 created_at ,  created_user   , delete_flag = 0 (tinyint)

2.如果是修改  寫入資料 updated_at , updated_user

3.如是是刪除  寫入資料 delete_flag = 1   delete_at  ,   delate_user  （主要目的是更改 flag 不直接上傳資料 ）


created_at                         updated_at                     created_user              updated_user   delete_flag    delete_at    delate_user

2018-03-10 23:00:00          2018-03-10 23:00:00        A123456                     A258476         0                   none          none



在每個列表裡 select 增加  delete_flag = 0 判斷  （ 已經刪除的資料就不要顯示 ）
